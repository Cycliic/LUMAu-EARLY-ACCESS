# LUMAu---Minecraft-Mod-Manager
A easy Mod Manager for Minecraft to Manage Mods. 

### About

The program was made my Cyclic, you can install and uninstall
custom mods onto Minecraft automatically.

This program has been made in Visual Studio by its creators. Discord message me to add other mods onto the program

### All Mods
The mods will be made by its creators, do not distribute!
 - Essential
 - Fabric API
 - Just Enough Items
 - GeckoLib
 - Cloth Config API
 - Architectury API
 - Mouse Tweaks
 - AppleSkin
 - Balm
